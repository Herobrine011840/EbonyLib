# EbonyLib is a library for Forge 1.20.1 that focuses on adding new ways to configure custom Features and Jigsaw structures! You can find examples of how to use these worldgen features in JSON in the repository.

Curseforge page: https://www.curseforge.com/minecraft/mc-mods/ebonylib

Modrinth Page: https://modrinth.com/mod/ebonylib

# The features of this mod currently include:

## -Jigsaw structure types that don't have the normal 7 levels/128 blocks from center limits

## -Jigsaw structure types that can check for a solid block before placing

## -Banded Ore features that can place large discs of blocks in layered bands

## -Glow Lichen-like growths that place more than 2 blocks of that kind along any solid face

## -And even new, highly customizable Trunk Placers for custom trees!
